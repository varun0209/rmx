Rule Setup
  • Condition: Object -> Property -> Operator -> value (Device.Step = “2100”)
  • Multiple conditions can be append with OR / AND (Device.Step = “2100” AND Device.OEM_CD = “APPLE”)
  • Adding Multiple Rules:
    1. Group the conditions with “(“ and “)” - 1st rule
    2. Append OR
    3. Add another group of Conditions – 2nd rule
    4. Example: (Device.Step = “2100” AND Device.OEM_CD = “APPLE”) OR (Device.Step = “3140” AND Device.OEM_CD = “LG”)
